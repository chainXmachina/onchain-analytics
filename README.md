# Onchain Analytics

This repository contains on-chain data analysis using Dune SQL, focusing on blockchain activity, user behavior, and tokenomics insights.

## What’s inside

- Dune SQL queries
- Dashboard links
- Simple analyses of on-chain data
- Experiments with blockchain metrics

## Focus Areas

- Token transfers and activity
- Wallet behavior
- Protocol usage
- Basic tokenomics indicators

## Tools

- Dune Analytics (SQL)
- Blockchain datasets (Ethereum and others)

## Dashboards

- https://dune.com/workspace/u/chainxmachina/activity

## Example Analyses

- Daily active wallets
- Token transfer volume
- User activity trends

## Goal

The goal of this repository is to develop practical skills in onchain analytics and build a foundation for tokenomics and economic system analysis.
